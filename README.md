# Jenkins Tutorial

A comprehensive tutorial and learning repository for Jenkins CI/CD pipeline configuration and automation.

## 📋 Project Overview

This repository contains practical examples, configurations, and tutorials for working with Jenkins. Whether you're new to Jenkins or looking to enhance your CI/CD pipeline skills, you'll find valuable resources here.

## 🛠️ Technology Stack

This project is built with:

- **HTML** (78.3%) - Web-based interface and documentation
- **JavaScript** (10%) - Interactive components and client-side scripts
- **Python** (9.2%) - Backend automation and scripting
- **Dockerfile** (2.5%) - Container configuration for Jenkins deployment

## 📁 Repository Structure

```
jenkins-tute/
├── HTML files/          # Web interface and documentation
├── JavaScript files/    # Interactive scripts and utilities
├── Python scripts/      # Automation and backend logic
├── Dockerfile          # Docker container setup
└── README.md           # This file
```

## 🚀 Getting Started

### Prerequisites

- Jenkins (latest version recommended)
- Docker (for containerized setup)
- Python 3.x
- A modern web browser

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/imTanush02/jenkins-tute.git
   cd jenkins-tute
   ```

2. **Using Docker:**
   ```bash
   docker build -t jenkins-tute .
   docker run -p 8080:8080 jenkins-tute
   ```

3. **Manual Setup:**
   - Refer to individual configuration files for manual setup instructions
   - Follow the HTML documentation for detailed guides
